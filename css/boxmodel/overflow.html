overflow 속성 - visible (default 속성)

요소의 박스에 내용이 더 길때 어떻게 보일지를 선택하는 요소이다.
- visible : 자식 요소가 부모 요소의 범위를 넘어가도 그대로 보여준다.
- scroll : 스크롤바가 추가되어 스크롤 할 수 있다.
- hidden : 부모 요소의 범위를 넘어가는 자식 요소의 부분은 보이지 않는다.
- auto : 내용이 넘치지 않으면 그대로 표시하고, 내용이 넘치면 스크롤바를 표시한다.
